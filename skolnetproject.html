Network Design Project Overview

Objective:

Enhance connectivity, ensure security, and manage content access in a school setting according to UK compliance standards.

Components:

Router:

Installed a high-performance router to manage the incoming internet connection from the ISP, acting as the gateway for all network traffic. Configured with robust firewall settings to secure the network perimeter.

Switch:

A managed network switch was deployed to connect all wired devices within the school, including servers, desktop PCs, and network endpoints. This switch provides VLANs for segmenting the network (e.g., student network, staff network, guest network).

WiFi Endpoints:

Four WiFi access points (APs) were strategically placed around the school to provide comprehensive wireless coverage. These APs were configured to broadcast multiple SSIDs for different user groups, ensuring that guest access did not interfere with educational activities.

Centralized DNS Server:

  • Set up a local DNS server to handle domain name resolution.
  • Crucial for DNS filtering to block access to unauthorized websites, ensuring compliance with educational content standards.
  • Logging and monitoring web traffic for security and compliance reporting.

Screen Monitoring and Management Software:

  • Installed on over 20 school PCs, this software allows:
  • Remote monitoring of student computer screens by teachers or admin staff.
  • Management capabilities like locking screens, pushing notifications, or initiating software updates across multiple machines simultaneously.

Installation Process:

  • Site Survey: Conducted a preliminary survey to determine optimal locations for WiFi endpoints for maximum coverage and performance.
  • Cabling and Infrastructure: Laid new or utilized existing network cabling to connect the switch, router, and WiFi endpoints. Ensured all hardware was securely mounted and accessible for maintenance.
  • Configuration:
    • Set up the router with appropriate WAN settings, DHCP for IP address distribution, and initial firewall rules.
    • Configured the switch with VLANs to segment traffic, improving network performance and security.
    • WiFi endpoints were configured with security protocols (WPA3), SSID settings, and band steering for efficient use of 2.4GHz and 5GHz bands.
  • Software Installation: Deployed screen monitoring software on school PCs, ensuring integration with the centralized management system for ease of use by teachers.
  • DNS Server Setup: Configured the DNS server with blacklists and filters for web content, integrated with UK educational standards for content filtering.
  • Testing and Optimization: Conducted thorough testing to ensure connectivity, speed, and security features were working as intended. Optimized the WiFi setup based on initial usage data to avoid congestion and ensure coverage in all learning areas.

Outcome:

  • Seamless internet access across the school with robust WiFi coverage.
  • Enhanced security through content filtering, ensuring compliance with educational standards.
  • Effective management of digital resources, allowing teachers to focus on teaching rather than IT management.
  • A scalable solution that can accommodate future growth or technological upgrades within the school.
